# 권한 관련 기능
### 1. 로그인/회원가입 기능
- JWT(JSON Web Token) 기반 토큰 인증 방식
- 비밀번호 해싱(bcrytp)
- 세션 관리

### 2. 역할기반 권한 관리(RBAC)
- 일반사용자 : 제품조회, 자신의 제품만 수정 삭제
- 매니저 : 일반 사용자와 동일한 권한
- 관리자 : 모든 제품 수정/삭제 가능

### 3. 소유자 검증
- 제품 수정/삭제 시 소유자 확인
- 관리자는 모든 제품에 대한 권한 보유
- UI에서 권한에 따른 버튼 표시/숨심

### 4. UX (사용자 경험 개선)
- 네비게이션 바에 현재 사용자 정보 및 역할 표시
- 권한에 따른 메뉴 접근 제어
